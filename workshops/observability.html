Five Steps to Observable Software

The workshop is built around a simple, iterative loop. No more vague Jira tickets like "add logging" — instead, answer specific, interesting questions by adding Just Enough Instrumentation.

  • Question — Form the question you are trying to answer.
  • Decide Data — Decide on which data is needed to answer it.
  • Build — Write code to instrument your app and collect the data.
  • Use — Use your observability tool — logs, traces, metrics — to try to answer the question.
  • Reflect — Did you answer the question? If not, return to the relevant step and iterate.
